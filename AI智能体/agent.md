## 销售客服案例

案例实操： 

### **1.进入工作空间： 创建知识库。** 

知识库的上传后设置，定义如何处理你上传的文档内容。 这里案例为{ Q A}问答模式。

```
*手机*
{
"Q:1.小熊猫牌手机的电池容量和充电速度如何?",
"A: HUAWEL nova 13配备了500 mAh(典型值)的电池，电池额定值为4900 mAh，该手机支持最大有线充电功率100 W，并且具备华为超级快充Turpbo模式，在实验室条件下，使用原装华为超级快充充电套装，可以在10分钟内将电量从3%至50%。实际使用中，充电速度可能会因产品个体差异、使用习惯和环境因素有所不同。"
 }
```

选择分段策略，自定义模式。  符合为{ ，分段最大长度：是说截取一段数据的最大字符数。 重叠度10%，是避免分段内容前后段丢失，提前重叠一点保存数据。

### 2**创建智能体环节**

介绍智能体的功能，将会展示给智能体的用户.  **重要环节***

```
手机、平板、电脑问题别发愁，不管是售前挑选，还是售后遇到使用问题，都能给你靠谱的解答和实用的建议，让你买得明白，用得省心。
```

### **3进入编排界面**- 创建工作流

  创建变量 ， 变量分为用户变量，系统变量。

用户变量：回复语言 中文， 语气 友好。  

系统变量：获取的经纬度，用户的id.

##### 工作流和对话流的区别

工作流： 问一次，执行一次，没有对话历史。

对话流：有对话历史。



先选择工作流 ： 命名  dianshagnkefu

重点： 工作流表述，告诉大模型什么时候调用。 例如：如下

```
当用户咨询手机/笔记本电脑/平板的相关问题时，调用工作流
```



#### **进入添加节点**

节点按照功能，划分为6类。   



这里选择一个知识库检索节点， 前提是自己已经上传了知识库。节点选择后，点击知识库节点，为它选择知识库，并设置大模型查找知识库的模式，和返回数据量默认5。 参数的大小有利弊，查看参数的提示语。

  返回数量默认5，那么该节点模型，会查找匹配度最高的5条，以数组形式返回。



**知识库节点后 + 大模型节点**。 目的是把知识库里检索出来的数据，经大模型进行文字润色和语言重组为通顺的句子。



大模型使用方法：

  a.输入的变量。

b. 系统提示词*

​    关键点：为明星指定你是谁，干什么，怎么样。 还可以使用变量，a环节输入的变量。 {{ 变量}}

```

根据用户的问题和知识库返回数据
#角色
你是一个电商客服，负责回复用户问题。你会接收4个输入:
1.用户的问题
2.知识库返回的数据

4.回复的语气

### 技能 1:问题理解

1.仅回答与电商相关的问题，不回答无关话题。
2.尽量使用清晰简练的语言来回答用户的问题,整个回答过程中，始终以用户的需求为中心。
```
