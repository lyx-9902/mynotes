## 备案网站

https://ywtb.mps.gov.cn/

填写地址：

https://beian.mps.gov.cn/



个人博客网站需在开通后 30 日内完成公安备案，核心流程为：先完成工信部 ICP 备案，再在全国互联网安全管理服务平台注册并实名认证，填写主体与网站信息，提交审核，通过后在网站底部添加备案号与图标。

------

### 备案前提与入口

- **前置条件**：已取得工信部 ICP 备案号（[beian.miit.gov.cn](https://beian.miit.gov.cn/)）。
- **备案入口**：全国互联网安全管理服务平台（[beian.mps.gov.cn](https://beian.mps.gov.cn/)）。
- **时限要求**：网站开通后 30 日内完成备案，逾期可能被处罚。

------

### 详细步骤（含材料与填写要点）

1. 注册与实名认证

   - 访问平台注册账号，完成手机验证与人脸识别（部分地区需下载 “公安备案” APP 辅助）。
   - 进入 “个人中心”→“信用认证”，选择 “个人” 主体，填写姓名、身份证号，上传身份证正反面及手持身份证照片（确保清晰、无遮挡）。

2. 填写开办主体信息

   - 确认主体信息无误后提交，等待信用认证通过（一般 1-2 个工作日）。

3. 新增网站备案申请

   - 进入 “新办网站申请”，选择已认证的个人主体。
   - 填写网站基本信息：
     - 网站名称：与 ICP 备案一致，避免敏感词。
     - 域名：填写主域名及所有子域名，上传域名证书（从域名注册商处获取）。
     - 服务器信息：填写公网 IP、接入商名称及 ID（从云服务商后台获取）。
     - 网站服务类型：选择 “个人博客 / 资讯”，不涉及论坛、电商等需前置许可的内容。
     - ICP 备案号：准确填写，如 “豫 ICP 备 XXXXXXXX 号”。
   - 填写负责人与联络人信息（与主体一致即可）。

4. 提交审核与结果查询

   - 确认所有信息无误后提交，等待公安机关审核（约 3-15 个工作日）。
   - 审核通过后，在 “已备案网站” 中查看备案号（如 “豫公网安备 XXXXXXXX 号”）。

5. 添加备案标识至网站

   - 下载备案号图标，获取 HTML 代码，将图标与备案号添加至网站首页底部显著位置（如页脚）。

   - 示例代码（替换为实际编号）：

     html

     ```html
       <img src="备案图标地址" alt="公安备案"/>豫公网安备XXXXXXXX号
     </a>
     ```

     

   - 确保标识可正常访问且永久有效。

------

### 常见问题与注意事项

- **照片审核失败**：确保照片清晰、光线充足、身份证信息完整，手持身份证时面部与证件均可见。
- **域名验证失败**：确认域名已完成 ICP 备案，且解析至备案的服务器 IP。
- **接入商信息错误**：准确填写云服务商名称（如阿里云、腾讯云）及提供的接入商 ID。
- **备案号未显示**：检查 HTML 代码是否正确，确保图标与链接可访问，建议使用浏览器开发者工具调试。

------

### 审核与后续管理

- **审核周期**：3-15 个工作日，以平台通知为准。
- **变更管理**：域名、服务器或负责人信息变更时，需在平台及时更新备案信息。
- **年度核查**：部分地区需参与年度备案核查，确保信息持续有效。

------

### 材料清单（提前准备）

| 材料类型         | 具体要求                                                     |
| ---------------- | ------------------------------------------------------------ |
| **个人证件**     | 身份证正反面照片、手持身份证照片（清晰无遮挡）               |
| **域名证明**     | 域名证书（有效期内，显示域名、注册商及持有人信息）           |
| **服务器信息**   | 公网 IP 地址、接入商名称及 ID（从云服务商后台获取）          |
| **ICP 备案信息** | ICP 备案号及备案截图（[beian.miit.gov.cn](https://beian.miit.gov.cn/)查询） |
