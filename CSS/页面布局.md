# 页面布局

屏幕：  1920 *1080

```
    <div class="father">
        <div class="box1"></div>
        <div class="box2"></div>
        <div class="box3"></div>
    </div>
        html,
        body {
            height: 500px;
            width: 500px;
        }

        .father {
            height: 100%;
            width: 100%;
            display: flex;
            flex-direction: column;
            border: 1px solid red;
        }

        .box1 {
            background-color: #F5F4A7;
        }

        .box2 {
            background-color: #D2A7F5;

        }

        .box3 {
            background-color: #90C7F2;
            flex: 1;
            overflow-y: auto;
        }
    </style>
```

第一种情形

```
父盒子 弹性盒子，轴向竖向。
box1 box2 内容撑开， box3占据剩余空间。
问题： 由于box3内容较多，超出父盒子的高度。
处理手段：   
1.  box3: 单独加一个  overflow-y: auto;

2. height:0;felx:1;
```
